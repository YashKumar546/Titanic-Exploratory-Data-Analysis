# 🚢 Titanic Exploratory Data Analysis

An Exploratory Data Analysis (EDA) project on the Titanic dataset using Python. This project demonstrates how to inspect, visualize, and understand a real-world dataset through statistical summaries, data visualization, and automated profiling.

---

## 📌 Project Overview

The objective of this project is to analyze the Titanic dataset and uncover meaningful patterns related to passenger demographics, survival rates, and data quality using Exploratory Data Analysis (EDA).

The project includes:
- Data exploration
- Summary statistics
- Missing value analysis
- Data visualization
- Correlation analysis
- Automated HTML profiling report

## 📈 Key Insights

- The dataset contains **891 passenger records** and **12 features**.
- The **Age** and **Cabin** columns contain missing values.
- Most passengers traveled in **Third Class (Pclass = 3)**.
- Male passengers outnumber female passengers.
- Female passengers had a higher survival rate.
- Fare values are positively skewed with several outliers.
- No duplicate records were found.
- Automated profiling provides additional statistical and correlation insights.
